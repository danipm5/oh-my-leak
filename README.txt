1. GET /api/users - Retrieves a list of all users
2. POST /api/users - Creates a new user
3. GET /api/users/{id} - Retrieves a specific user by ID
4. PUT /api/users/{id} - Updates a specific user by ID
5. DELETE /api/users/{id} - Deletes a specific user by ID
6. GET /api/posts - Retrieves a list of all posts
7. POST /api/banderin - Health status (dev)
8. GET /api/posts/{id} - Retrieves a specific post by ID
9. PUT /api/posts/{id} - Updates a specific post by ID
10. DELETE /api/posts/{id} - Deletes a specific post by ID
